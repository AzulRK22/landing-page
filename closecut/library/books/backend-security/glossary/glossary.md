# Appendix T. Glossary

**Active membership:** A Circle member document whose status permits current access.  
**Callable Function:** Authenticated Firebase HTTPS function invoked through the client SDK.  
**Circle:** A private, membership-scoped sharing space.  
**Conflict:** Divergent local and remote states requiring deterministic reconciliation.  
**DTO:** Remote transport representation mapped to and from domain/local models.  
**Idempotency:** Repeating an operation has the same logical effect as performing it once.  
**Local source of truth:** SwiftData state used for immediate UI behavior.  
**Pending action:** Durable local representation of a remote mutation not yet confirmed.  
**Principal:** Authenticated Firebase UID used for authorization.  
**Soft deletion:** Logical deletion represented by a tombstone such as `deletedAt`.  
**Tombstone:** Persisted marker that prevents deleted data from being resurrected during sync.  
**Trusted Circle:** Product term for explicitly selected, membership-controlled sharing context.
