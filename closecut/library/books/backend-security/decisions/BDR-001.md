# BDR-001 - SwiftData as UI Source of Truth

**Status:** Accepted

## Context

CloseCut requires a stable platform decision for swiftdata as ui source of truth.

## Decision

The UI reads and writes local SwiftData models first. This preserves responsiveness, offline capability, and deterministic ownership of immediate user state.

## Consequences

All implementation and migration work must preserve this contract until superseded by an approved record.
